This folder contains the Test split simulation #0000 from the full dataset.

At this time, we are unable to provide links to the full dataset. The dataset can be created using the script `main_sims.py`. 

Alternatively, the contents of this folder can be placed in your dataset folder. E.g.,
    - Set up your local system following instruction in the main README, ensuring that the `dataset_root` folder exists in your local_system yaml.
    - Move or copy the I_512_1-1 folder (and its contents) to that dataset_root directory.
